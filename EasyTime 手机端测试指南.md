## 📲 PWA 安装测试

### iOS (Safari)
1. 打开网站 `http://192.168.1.24:5173`
2. 点击底部**分享按钮** (□↑)
3. 向下滚动，选择 **"添加到主屏幕"**
4. 点击右上角 **"添加"**
5. 从主屏幕启动应用

### Android (Chrome)
1. 打开网站 `http://192.168.1.24:5173`
2. 点击右上角 **⋮ 菜单**
3. 选择 **"安装应用"** 或 **"添加到主屏幕"**
4. 点击 **"安装"**
5. 从主屏幕启动应用

## 🔧 方法二：使用 ngrok（适合远程测试）

如果你需要在不同网络下测试，可以使用 ngrok：

### 安装 ngrok
```bash
# macOS
brew install ngrok

# 或访问 https://ngrok.com 下载
```

### 使用步骤
```bash
# 1. 确保开发服务器运行中
npm run dev

# 2. 在新终端窗口运行
ngrok http 5173
```

### 访问方式
ngrok 会生成一个公网 URL，例如：
```
https://xxxx-xxxx-xxxx.ngrok.io
```
在任何设备上都可以访问这个地址！

## 🐛 常见问题

### 1. 无法访问局域网地址

**问题**: 手机无法打开 `http://192.168.1.24:5173`

**解决方案**:
- 确认手机和电脑在同一 WiFi
- 检查电脑防火墙设置
- 尝试关闭 VPN
- 重启开发服务器：
  ```bash
  # 停止服务器 (Ctrl+C)
  npm run dev
  ```

### 2. IP 地址变化

**问题**: 电脑 IP 地址可能会变化

**解决方案**:
- 每次测试前重新获取 IP：
  ```bash
  ifconfig | grep "inet " | grep -v 127.0.0.1
  ```
- 或在路由器中设置静态 IP

### 3. PWA 不能安装

**问题**: 没有看到安装提示

**解决方案**:
- **iOS**: 必须使用 Safari 浏览器
- **Android**: 推荐使用 Chrome
- 确保网站已完全加载
- 查看浏览器地址栏是否有安装图标

### 4. 虚拟键盘遮挡内容

**问题**: 输入时底部导航被键盘遮挡

**解决方案**:
- 已优化：输入框聚焦时会自动调整
- 如有问题，尝试滚动页面
- 或切换到横屏模式

### 5. 主题切换不生效

**问题**: 切换主题后没有变化

**解决方案**:
- 刷新页面
- 清除浏览器缓存
- 检查系统主题设置（如选择"跟随系统"）

## 📊 性能优化建议

### 开发时优化
1. 使用 Chrome DevTools 远程调试
   - 电脑 Chrome → More Tools → Remote Devices
   - USB 连接手机（Android）

2. 使用 Safari 开发者工具
   - Mac Safari → 开发 → [你的iPhone]
   - 需开启 iPhone 的 Safari 开发者选项

### 网络优化
- WiFi 信号强度影响加载速度
- 靠近路由器测试
- 避免多设备同时占用带宽
